Joke Generator- Machine Learning

Note: This algorithm is generating jokes that can be offensive. It would be ideal to have an algorithm produce non-offensive jokes. However, I take this obstacle to be representitive of how not all data is pure. Also this is symbolic of the way humans learn and perpetuate discriminatory beliefs. If a person was taught sexist and racist beliefs, they would most likely carry those beliefs onto the next generation like how the algorithm capitalized on the racist and sexist jokes that were apart of the 200,000+ datatset.

The following jokes are representitve of what the computer came up with. You may note that some jokes are actually funny, some are offensive, some don't make sense, and some are incomplete. These are all distinguishing elements of the machine learning process.
